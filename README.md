# Financial News Aggregator Bot

## Overview
Automation system that collects, filters and summarizes daily financial and economic news from multiple sources.

## Core Features
- Data collection from 6 financial news portals
- Filtering relevant economic topics
- Ranking and selecting Top-10 daily news
- Automated daily digest delivery via Telegram

## Tech Stack
- n8n
- Web scraping / RSS logic
- Telegram Bot API
- Structured filtering logic

## System Logic
1. Fetch news from predefined sources
2. Filter by keywords and topic relevance
3. Remove duplicates
4. Rank by importance
5. Send Top-10 structured summary

## Engineering Challenges
- Handling inconsistent data formats
- Removing duplicate content
- Designing ranking logic
- Ensuring stable daily execution

## Result
Fully automated daily financial news briefing system.
